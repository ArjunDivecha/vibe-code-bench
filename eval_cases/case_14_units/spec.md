# Build me a Unit Converter

Create a unit conversion tool:

**Categories:**
- Length (km, m, cm, mm, miles, feet, inches)
- Weight (kg, g, lb, oz)
- Temperature (C, F, K)
- Volume (L, mL, gallons, cups)
- Time (hours, minutes, seconds)

**Features:**
- Select category, then from/to units
- Type in one field, other updates instantly
- Swap button to reverse conversion
- Show conversion formula used
- Common conversions as quick buttons

Clean, responsive design. Single HTML file.
