# Build me a Typing Speed Test

Create a typing test web app:

- Show a paragraph of text to type
- Text input box below it
- Real-time highlighting of correct/incorrect characters
- Show WPM (words per minute) as you type
- Show accuracy percentage
- Timer counting up or fixed duration (30 seconds)
- When done, show final stats
- Button to try again with different text

Include like 10 different text samples to practice with. Single HTML file.
