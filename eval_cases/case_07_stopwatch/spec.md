# Build me a Stopwatch and Timer

Create a dual stopwatch/timer web app:

**Stopwatch mode:**
- Start, stop, lap, reset buttons
- Shows elapsed time in HH:MM:SS.ms format
- Lap times shown in a list

**Timer mode:**
- Set hours, minutes, seconds
- Countdown display
- Alert/sound when done
- Pause/resume

Add a toggle to switch between modes. Make it look like a digital clock display. Single HTML file.
