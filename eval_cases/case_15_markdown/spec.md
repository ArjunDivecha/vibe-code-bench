# Build me a Markdown Editor

Create a split-pane markdown editor:

- Left side: text input for markdown
- Right side: live preview of rendered HTML
- Support for:
  - Headers (# ## ###)
  - Bold (**text**) and italic (*text*)
  - Lists (- and 1. 2. 3.)
  - Links [text](url)
  - Code blocks ```code```
  - Blockquotes (>)
- Save/load from localStorage
- Dark mode toggle
- Export as HTML button

Single HTML file.
