# Slide Summary Reporter

A Python tool that converts text-based slide files into a beautiful HTML report with summary, table of contents, and printable cards.

## Usage

1. Create a directory with your slide text files (e.g., `slide1.txt`, `slide2.txt`, etc.)
2. Run the tool:
   ```bash
   python slide_summary_reporter.py /path/to/slide/files
   ```
3. Open the generated `report.html` in your browser
4. Use the "Print to PDF" button to save as PDF

## Features

- Automatically generates a summary from the first few lines of each slide
- Creates a table of contents with links to each slide
- Responsive card layout for slides
- Print-friendly CSS with proper page breaks
- Premium typography and styling
- Smooth scrolling navigation

## Requirements

- Python 3.x (standard library only)
- No external dependencies
