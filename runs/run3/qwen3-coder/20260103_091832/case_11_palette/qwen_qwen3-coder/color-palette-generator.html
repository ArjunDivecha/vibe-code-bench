Color Palette Generator

Create beautiful color combinations with a single click

How to Use

  • Click "Generate Palette" or press Spacebar to create a new palette
  • Click the lock icon on any color to lock/unlock it
  • Click on a color to copy its HEX code to clipboard
  • Save your favorite palettes using the "Save Palette" button
