# Slide Summary Reporter

A Python tool that generates beautiful HTML reports from slide text files.

## Usage

1. **Create slide text files** (e.g., `slide1.txt`, `slide2.txt`, etc.)
   - First line: Slide title
   - Remaining lines: Slide content

2. **Run the reporter:**
   ```bash
   python slide_reporter.py
   ```
   
   Or specify a directory:
   ```bash
   python slide_reporter.py /path/to/slides
   ```

3. **Open `report.html`** in your browser

4. **Print to PDF** using the button or Ctrl+P/Cmd+P
